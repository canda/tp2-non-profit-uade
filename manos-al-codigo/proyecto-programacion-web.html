Programación web

Mi primera página

Una experiencia para crear una página web propia desde cero, aprendiendo HTML, CSS y diseño responsable a partir de una causa que cada participante elige.

Apoyar este proyecto Volver a proyectos

Qué hicieron

Diseñaron bocetos, organizaron contenido, escribieron sus primeras etiquetas HTML y aplicaron estilos para contar una idea con claridad.

Resultados

Los sitios finales se compartieron con familias y organizaciones del barrio, convirtiendo el aprendizaje en una pieza real de comunicación.

Aprendizajes

  • Estructura básica de una página web.
  • Diseño visual simple, claro y accesible.
  • Comunicación de una causa cercana usando tecnología.
